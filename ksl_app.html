Kenyan Sign Language Recognition Model

Python | Jupyter Notebook

Project Portfolio

Project Brief

This project is a deep learning model that uses the Long Short Term Memory algorithm to map hand gestures to their corresponding Kenyan Sign Language word. The solution was developed using a development procedure relevant to machine learning or data science projects.

This project explores how deep learning solutions can be implemented on real-world problems to effect social change, especially for persons with disabilities in Kenya.

Technologies Used - Python, Jupyter Notebook

QuizTime Mobile Application

Figma | Flutter | Dart | Firebase

Project Portfolio

Project Brief

This project is a pop-quiz mobile application implementing an array of quizzes from different categories availed for user selection. The project was part of my coursework's final semester project which required developing a fully functional mobile application.

The QuizTime mobile application was designed and developed using Figma for the mobile application's user interface. Front end components were programmed using The Flutter Frawework's Dart Programming Language with Google's Firebase the basis for setting up backend features.

Technologies Used - Figma, Flutter, Dart, Firebase
